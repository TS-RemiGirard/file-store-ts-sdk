# file-store-ts-sdk

Authenticate and use [file-store](https://github.com/TS-RemiGirard/file-store).

## Usage

Install with

```bash
  npm install @toosmartdev/file-store-ts-sdk
```

See test/e2e.test.ts for detailed usage.
